DUCK HUNT

GitHub

• Duck Hunt CE is a clone of the classic NES Duck Hunt for the TI-84 Plus CE calculator.
• The ducks fly in different patterns, and the player must shoot them before they fly out of the screen.
• The player can earn points for shooting ducks, and the game ends when all of the ducks have been shot.
• Duck Hunt CE is a fun and challenging game that can be enjoyed by players of all ages.

GFX3

GitHub

• GFX3 is a sprite stacking library for the TI-84 Plus CE toolchain.
• It makes it easy to create and handle stacked sprites in CE Toolchain projects.
• Developers can layer multiple sprites on top of each other, change their location, size, and rotation, and even make them translucent.
• GFX3 can be used to create more complex and dynamic visuals in projects.

Xenon

GitHub

• Xenon is a command line and graphical user interface (GUI) shell for the TI-84 Plus CE calculator.
• It provides a user-friendly interface for interacting with the calculator's operating system.
• With Xenon, users can navigate the file system, run programs, and perform system tasks.
• The shell includes features such as command history, tab-completion, and support for piping and redirecting input and output.
• Additionally, Xenon has a set of built-in utilities that enhance the functionality of the calculator by providing additional tools and features.
• The shell is optimized to work within the memory and processing constraints of the calculator.

Neural

GitHub

• Neural is a coevolutionary network for the CE calculator that detects numbers using the MNIST dataset. • The MNIST dataset is a set of handwritten digits that is commonly used to train machine learning models. • Neural uses a coevolutionary algorithm to evolve a network of interconnected nodes that can detect numbers in the MNIST dataset. • The network was developed as part of a research project to explore the use of coevolutionary algorithms for developing machine learning models on the CE calculator.
